  <action>
    Per CONTEXT.md: inline Monaco diff, per-hunk buttons, red/green style, discard rejected.

    1. Add to src/shared/types.ts:
       ```typescript
       export interface DiffHunk {
         id: string;
         startIndex: number;
         endIndex: number;
         type: 'add' | 'delete' | 'replace';
         originalLines: string[];
         modifiedLines: string[];
         status: 'pending' | 'accepted' | 'rejected';
       }

       export interface PendingDiff {
         id: string;
         filePath: string;
         originalContent: string;
         modifiedContent: string;
         hunks: DiffHunk[];
         toolCallId: string;
         timestamp: number;
       }
       ```

    2. Add to src/shared/constants.ts:
       - `MAX_DIFF_FILE_LINES = 5000` — files with more lines skip inline diff, show in tool card only
       - `DIFF_CONTEXT_LINES = 3` — context lines around each hunk

    3. Create src/renderer/stores/diff-store.ts (Zustand):
       - State:
         - `pendingDiffs: PendingDiff[]`
         - `activeDiffId: string | null`
       - Actions:
         - `addDiff(diff: PendingDiff)` — compute hunks, add to pendingDiffs, set activeDiffId
         - `acceptHunk(diffId: string, hunkId: string)` — mark hunk accepted, apply to disk via IPC
         - `rejectHunk(diffId: string, hunkId: string)` — mark hunk rejected, do not apply
         - `acceptAll(diffId: string)` — apply all pending hunks, remove from pendingDiffs
         - `rejectAll(diffId: string)` — reject all hunks, remove from pendingDiffs
         - `clearDiffs()` — clear all pending diffs
         - `setActiveDiff(diffId: string | null)` — set active diff for Monaco display
       - Hunk computation: simple line-by-line comparison splitting into contiguous change regions with context. Use a basic diff algorithm (Myers or patience diff from `diff` npm package, or implement simple LCS-based diff).

    4. Add IPC channels for applying diffs:
       - `file:apply-hunk` — request: { filePath, originalContent, modifiedContent, hunksToApply: string[] }, response: { success: boolean }
       - Add to ipc-channels.ts, preload, ipc-handlers

    5. Write tests in src/renderer/stores/__tests__/diff-store.test.ts covering all actions.
