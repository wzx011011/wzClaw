    Create `src/main/persistence/session-store.ts` with the SessionStore class:

    1. **Constructor**: Takes `workspaceRoot: string`. Computes `projectHash` as first 16 hex chars of `crypto.createHash('sha256').update(workspaceRoot).digest('hex')`. Sets `sessionsDir` to `path.join(app.getPath('userData'), 'sessions', this.projectHash)`. Creates the directory with `fs.mkdirSync(this.sessionsDir, { recursive: true })`.

    2. **SessionMeta interface** (exported):
    ```typescript
    export interface SessionMeta {
      id: string
      title: string
      createdAt: number
      updatedAt: number
      messageCount: number
    }
    ```

    3. **appendMessage(sessionId: string, message: ChatMessageLike): void**: Appends one JSON line to `sessionsDir/{sessionId}.jsonl` using `fs.appendFileSync`. The `ChatMessageLike` type is `{ role: string; content: string; timestamp: number; id?: string; toolCalls?: unknown[]; usage?: { inputTokens: number; outputTokens: number } }`.

    4. **appendMessages(sessionId: string, messages: ChatMessageLike[]): void**: Calls appendMessage in a loop. Used after agent:done to persist all messages from the completed turn.

    5. **loadSession(sessionId: string): ChatMessageLike[]**: Reads the JSONL file. Splits by newline. For each non-empty line, try/catch JSON.parse. On parse failure, log a warning with `console.warn` and skip the line (PERSIST-05). Returns array of parsed objects. Returns empty array if file does not exist.

    6. **listSessions(): SessionMeta[]**: Reads `sessionsDir` for `*.jsonl` files. For each file:
       - Read first line for title extraction: parse first JSONL line, if `role === 'user'` then `content.substring(0, 50)` + "..." if longer, else "Untitled"
       - Get file stats for createdAt (birthtime) and updatedAt (mtime)
       - Count lines for messageCount
       Sort by updatedAt descending (most recent first).

    7. **deleteSession(sessionId: string): boolean**: Deletes `sessionsDir/{sessionId}.jsonl`. Returns true if file existed and was deleted, false otherwise. Use `fs.unlinkSync`.
