Per D-41 (VS Code layout), D-44 (resizable panels), D-49 (Zustand for tabs), D-53 (VS Code Dark+ theme), D-55 (status bar):

1. **Install npm packages:**
   ```
   npm install @monaco-editor/react monaco-editor allotment
   ```
   Note: allotment is a lightweight split-pane library (React, resizable). react-window is deferred until we have performance data (D-46 says virtualized but that is optimization, not MVP). allotment replaces react-split-pane (simpler API, better maintained).

2. **Create src/renderer/stores/workspace-store.ts:**
   Zustand store for workspace state:
   - State: `rootPath: string | null`, `tree: FileTreeNode[]`, `isLoading: boolean`, `error: string | null`
   - Actions:
     - `openFolder()` — calls `window.wzxclaw.openFolder()`, sets rootPath, loads root tree
     - `loadTree(dirPath?: string, depth?: number)` — calls `window.wzxclaw.getDirectoryTree(...)`, replaces or merges into tree
     - `expandNode(dirPath: string)` — loads children for dirPath with depth=1, merges into tree at the right position (lazy loading per D-47)
     - `collapseNode(dirPath: string)` — removes children from tree node
     - `updateNode(path: string, changes: Partial<FileTreeNode>)` — find and update a node in the tree
     - `handleFileChange(filePath: string, changeType: string)` — update tree for created/modified/deleted files

3. **Create src/renderer/stores/tab-store.ts:**
   Zustand store for editor tabs per D-49:
   - State: `tabs: EditorTab[]`, `activeTabId: string | null`
   - Actions:
     - `openTab(filePath: string, fileName: string, content: string, language: string)` — if tab with filePath exists, activate it; else create new EditorTab (id = uuid), add to tabs, set active
     - `closeTab(tabId: string)` — remove tab, if was active, activate adjacent tab (prefer right, then left)
     - `setActiveTab(tabId: string)` — set activeTabId
     - `updateTabContent(tabId: string, content: string)` — update content, compute isDirty = (content !== diskContent) per D-50
     - `saveTab(tabId: string)` — calls `window.wzxclaw.saveFile({ filePath, content })`, then updates diskContent = content, isDirty = false
     - `getActiveTab(): EditorTab | undefined` — getter
     - `refreshTabContent(tabId: string, newContent: string)` — update both content and diskContent (used for agent edits per D-52)

4. **Create src/renderer/styles/ide.css:**
   CSS for the IDE layout. Dark theme colors matching VS Code Dark+:
   - `--bg-primary: #1e1e1e` (editor background)
   - `--bg-secondary: #252526` (sidebar)
   - `--bg-tertiary: #333333` (tab bar, status bar)
   - `--text-primary: #cccccc`
   - `--text-secondary: #858585`
   - `--accent: #007acc` (selection, active)
   - `--border: #404040`
   - `--tab-active-bg: #1e1e1e`
   - `--tab-inactive-bg: #2d2d2d`
   - `--dirty-indicator: #c5c5c5` (white dot)

   Reset styles, full height/width layout, scrollbar styling. No external CSS framework.
