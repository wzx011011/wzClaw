       - description: "Edit a file by replacing an exact string match. Use old_string to specify the text to find and new_string for the replacement. The old_string must match exactly (including whitespace/indentation) and must be unique in the file."
       - requiresApproval: true (per D-32)
       - inputSchema: `{ path: string (required), old_string: string (required, exact text to find), new_string: string (required, replacement text) }`
       - Zod schema: `z.object({ path: z.string().min(1), old_string: z.string().min(1), new_string: z.string() })`
       - execute(): Read file with `fs.readFile`. Count occurrences of old_string in content:
         - If 0 matches: return `{ output: "old_string not found in file: {path}", isError: true }` (race condition protection per D-03)
         - If >1 matches: return `{ output: "old_string matches {count} times in file. Provide more context to make it unique.", isError: true }`
         - If exactly 1 match: replace old_string with new_string. Write back with `fs.writeFile`. Return `{ output: "Edited {path}: replaced {oldLen} chars with {newLen} chars", isError: false }`
       - This implements search-and-replace pattern per TOOL-03 requirement.

       - Zod schema: `z.object({ command: z.string().min(1), timeout: z.number().positive().optional() })`
       - execute(): Use `child_process.exec` with `{ cwd: context.workingDirectory, timeout: timeout ?? 30000, maxBuffer: 1024 * 1024 }`. On Windows, use `cmd.exe /c` prefix. Capture stdout and stderr. Return `{ output: stdout + (stderr ? "\nSTDERR:\n" + stderr : ""), isError: exitCode !== 0 }`. On timeout, return `{ output: "Command timed out after {timeout}ms", isError: true }`. Pass context.abortSignal: if aborted, kill the child process. Truncate output at MAX_TOOL_RESULT_CHARS.
